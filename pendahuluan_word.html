1. Manajemen File

Menyelamatkan hasil kerja keras Anda

Jangan sampai ketikan Anda hilang karena mati lampu atau salah simpan! Pahami perbedaan mendasar cara mengelola dokumen Anda.

  • Save vs Save As

    Save (Simpan): Menyimpan pembaruan pada dokumen yang sudah pernah disimpan sebelumnya (menimpa file lama secara langsung).
    Save As (Simpan Sebagai): Menyimpan dokumen menjadi file BARU, di lokasi folder baru, atau dengan nama yang berbeda. Sangat berguna untuk membuat salinan tanpa merusak file asli.

  • AutoSave (Penyimpanan Otomatis)

    Sebuah tuas di pojok kiri atas (khusus pengguna Microsoft 365 & OneDrive). Jika diaktifkan, Word akan menyimpan dokumen setiap beberapa detik secara otomatis. Mencegah bencana kehilangan data akibat laptop crash atau mati mendadak.

  • Format Dokumen: .DOCX vs .PDF

    Saat menggunakan Save As, pastikan formatnya benar.
    .DOCX: Gunakan ini jika dokumen masih butuh diedit.
    .PDF: Gunakan ini saat dokumen sudah "Final" dan akan dikirim ke orang lain (Dosen/Klien). PDF akan "mengunci" tata letak agar font dan gambar tidak berantakan saat dibuka di HP atau laptop lain.

Tips Versioning Sederhana:
Berhentilah menggunakan nama file Skripsi_Final_Beneran_Revisi_Terakhir.docx. Biasakan menggunakan penomoran versi (Versioning) yang rapi, seperti Skripsi_Bab1_v1.docx, dan jika ada perbaikan simpan sebagai (Save As) Skripsi_Bab1_v2.docx. Ini memudahkan Anda melacak dan kembali ke versi lama jika dibutuhkan.

2. Basic Editing Tools

Alat esensial untuk memanipulasi teks
  • Tiga Serangkai: Copy, Cut, dan Paste
    Copy (Salin)
    Teks asli tetap ada, Anda membuat duplikatnya.
    Cut (Potong)
    Teks asli MENGHILANG untuk dipindahkan.
    Paste (Tempel)
    Meletakkan hasil Copy/Cut ke tempat baru.
  • Find & Replace (Cari & Ganti)

    Berada di tab Home sebelah kanan. Fitur ini menyelamatkan hidup Anda jika Anda salah mengeja nama perusahaan di dokumen proposal setebal 100 halaman.

    Find what: Djakarta
    Replace with: Jakarta
    (Word otomatis mengganti ratusan kata 'Djakarta' jadi 'Jakarta' dalam 1 detik)
  • Undo dan Redo

    Undo (Panah ke kiri) membatalkan kesalahan fatal yang baru saja Anda buat (misal: tidak sengaja menghapus satu tabel penuh). Redo (Panah ke kanan) berfungsi mengembalikan tindakan yang baru saja di-Undo.

3. Penggunaan Styles

WAJIB UNTUK TINGKAT MENENGAH
Fondasi Menuju Tingkat Mahir: Kesalahan terbesar pemula adalah membesarkan judul bab hanya dengan menggunakan Bold dan Font Size 16. Ini membuat Anda TIDAK AKAN PERNAH bisa membuat Daftar Isi otomatis nantinya.
Cara Kerja Heading:

Styles memberitahu "otak" Word bahwa teks tersebut bukan paragraf biasa, melainkan sebuah struktur rangka (outline).

  • Heading 1: Gunakan untuk Judul Utama atau Judul Bab Besar (Misal: BAB I PENDAHULUAN).
  • Heading 2: Gunakan untuk sub-judul di bawah Bab (Misal: A. Latar Belakang).
  • Heading 3: Gunakan untuk anak sub-judul (Misal: 1. Tujuan Penelitian).
  • Normal: Gunakan untuk mengetik isi paragraf biasa.

Contoh Struktur Dokumen yang Benar:

[Heading 1] BAB 1 PENDAHULUAN

[Heading 2] A. Latar Belakang Masalah

[Normal] Keterampilan komputer di era digital merupakan sebuah kewajiban mendasar yang harus dimiliki...

4. Bullets & Numbering

Membuat daftar poin yang rapi

Sangat krusial untuk menyusun dokumen formal (seperti SOP, instruksi kerja, atau syarat dan ketentuan). Fitur ini memisahkan blok teks panjang menjadi poin-poin agar mudah dibaca oleh mata.

Bullets (Poin Simbol)

Gunakan jika urutan poin tidak mementingkan hirarki/langkah.

Syarat Pendaftaran:

  • Fotokopi KTP
  • Pas Foto 4x6
  • Formulir Pendaftaran
Numbering (Angka)

Gunakan jika poin merupakan langkah berurutan (SOP/Instruksi).

Cara Menyimpan File:

  1. Klik tab File
  2. Klik Save As
  3. Pilih folder tujuan
Tips Indentasi: Jika titik simbol/angka terasa meloncat terlalu jauh ke tengah kertas, letakkan kursor di teks tersebut lalu klik ikon Decrease Indent (di sebelah ikon numbering) untuk menggesernya kembali merapat ke batas margin kiri.
Tombol 'Ctrl' adalah sahabat terbaik Anda

5. Shortcut Keyboard Dasar

Efisiensi Kerja Ekstra Cepat

Orang profesional akan terlihat sangat lincah saat bekerja karena mereka mengurangi penggunaan Mouse untuk tugas-tugas dasar. Menghafal 10 kombinasi tombol (Shortcut) ini akan memangkas waktu kerja Anda secara drastis.

Aksi Pengeditan
Ctrl + C Copy (Salin)
Ctrl + X Cut (Potong/Pindah)
Ctrl + V Paste (Tempel)
Ctrl + Z Undo (Batal Tulis)
Ctrl + A Select All (Blok Semua)
Format & File
Ctrl + B Bold (Tebal)
Ctrl + I Italic (Miring)
Ctrl + U Underline (Garis Bawah)
Ctrl + S Save (Simpan Cepat)
Ctrl + P Print (Cetak Kertas)
