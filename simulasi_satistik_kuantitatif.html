Alur Proses Perhitungan Peneliti

Berikut adalah panduan alur perhitungan agar hasil di atas dapat direplikasi dan dihitung ulang oleh peneliti menggunakan perangkat lunak statistik standar (IBM SPSS dan Microsoft Excel).

Langkah di IBM SPSS
  1. Statistik Deskriptif: Buka menu Analyze > Descriptive Statistics > Descriptives... Masukkan seluruh variabel penelitian ke kotak Variable.
  2. Korelasi Pearson: Klik Analyze > Correlate > Bivariate... Pilih variabel, pastikan Pearson dicentang.
  3. Regresi Berganda: Klik Analyze > Regression > Linear... Masukkan Y ke kotak Dependent, dan X (serta M) ke kotak Independent(s). Lihat tabel Model Summary untuk R² dan tabel Coefficients untuk Beta.
  4. Uji Mediasi Sobel: Lakukan regresi dua kali: (1) X terhadap M, (2) X dan M terhadap Y. Catat nilai Unstandardized B dan Std. Error, lalu hitung menggunakan rumus Sobel Z-value.
Langkah di Microsoft Excel
  1. Persiapan: Aktifkan Data Analysis Toolpak melalui menu File > Options > Add-ins.
  2. Statistik Deskriptif: Buka tab Data > Data Analysis > Descriptive Statistics. Atau gunakan rumus manual =AVERAGE(range) dan =STDEV.S(range).
  3. Korelasi: Gunakan menu Data Analysis > Correlation. Atau gunakan rumus =PEARSON(range1, range2). Format cell ke Percentage (%) untuk melihat persentase hubungan.
  4. Regresi Linier: Klik Data > Data Analysis > Regression. Masukkan Range Y dan Range X. Hasil R Square dan Coefficients akan muncul di sheet baru.
