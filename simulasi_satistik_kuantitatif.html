Statistik Deskriptif

Mengukur nilai pemusatan (Mean) dan sebaran variasi data sampel (Standar Deviasi) dari setiap variabel.

x̄ = Σx / n
s = √ [ Σ(x - x̄)² / (n - 1) ]
Uji Korelasi Pearson

Mengukur kekuatan dan arah hubungan linier antara dua variabel secara langsung (berkisar antara -1 hingga 1).

r = Σ(x - x̄)(y - ȳ)
√ [ Σ(x - x̄)² × Σ(y - ȳ)² ]
Regresi Linier Berganda

Memodelkan pengaruh beberapa variabel bebas (X) dan perantara (M) terhadap variabel terikat (Y) ke dalam satu persamaan.

Y = α + β±X± + β²X² + ... + ε
R² = SSR / SST
Pengujian Hipotesis

Menganalisis kemaknaan pengaruh secara parsial/sendiri (Uji t) maupun secara simultan/bersama-sama (Uji F).

t = β / SE(β)
F = (R²/k) / [ (1 - R²)/(n - k - 1) ]
Uji Sobel (Mediasi)

Menghitung rasio statistik (Z-value) untuk melihat signifikansi efek tidak langsung melalui variabel perantara (M).

Z = a × b
√ [ (b² × SEa²) + (a² × SEb²) ]
