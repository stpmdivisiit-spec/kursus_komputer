Mail Merge (Surat Massal)

Microsoft Word + Microsoft Excel

Mencetak Ribuan Dokumen dalam Sekali Klik

Mail Merge adalah fitur otomatisasi yang menghubungkan kerangka dokumen di Microsoft Word dengan basis data penerima (biasanya di Microsoft Excel). Bayangkan Anda harus membuat sertifikat untuk 500 peserta seminar. Daripada mengetik nama satu per satu yang memakan waktu berhari-hari, Mail Merge dapat menyelesaikannya dalam hitungan menit.

Gambar 1: Konsep Mail Merge. Satu Template Word + Tabel Excel = Ratusan Dokumen Berbeda.

Langkah 1: Siapkan Data di Excel

Membuat Daftar Penerima (Data Source)

Sebelum membuka Word, Anda wajib memiliki daftar data yang akan dimasukkan ke dalam surat. Buatlah tabel sederhana di Microsoft Excel dan simpan filenya.

  • Baris Pertama Wajib Judul Kolom: Baris paling atas (A1, B1, C1) harus berisi nama kategori data, misalnya: Nama_Peserta, Alamat, Jabatan. Judul kolom ini nantinya akan menjadi kode variabel di Word.
  • Isi Data Tanpa Baris Kosong: Mulai baris kedua dan seterusnya, isi dengan data penerima yang sebenarnya. Jangan ada baris yang dikosongi di tengah-tengah data.
  • Simpan file Excel tersebut di folder yang mudah ditemukan (misal dengan nama Data_Undangan.xlsx). Tutup aplikasi Excel jika sudah selesai.

Langkah 2: Hubungkan Word dengan Excel

Membangun Jembatan Data

Buka Microsoft Word, ketikkan kerangka surat undangan Anda secara normal. Biarkan bagian nama tujuan dan alamat kosong. Kemudian hubungkan dengan file Excel tadi.

  • Pergi ke Tab Mailings di Ribbon.
  • Klik tombol Select Recipients (Pilih Penerima).
  • Pilih opsi Use an Existing List... (Gunakan daftar yang sudah ada).
  • Jendela pencarian file akan muncul. Cari dan pilih file Data_Undangan.xlsx yang sudah Anda buat di langkah pertama, lalu klik Open.
  • Jika muncul kotak dialog konfirmasi Select Table, pastikan sheet yang berisi data Anda terpilih (biasanya Sheet1$), lalu klik OK.

Langkah 3: Insert Merge Field

Memasukkan Kode Variabel Pengganti

Sekarang saatnya meletakkan data dari Excel ke posisi yang tepat di dalam dokumen Word Anda.

  • Tempatkan kursor berkedip Anda tepat di tempat nama akan muncul (misalnya setelah kata "Kepada Yth. ").
  • Pada Tab Mailings, klik tombol Insert Merge Field.
  • Akan muncul daftar judul kolom dari Excel Anda (seperti Nama_Peserta, Alamat). Klik Nama_Peserta.
  • Sebuah kode kurung siku ganda akan muncul di lembar kerja Anda: «Nama_Peserta». Ini adalah penanda bahwa teks tersebut akan berubah secara dinamis.
  • Ulangi langkah ini untuk data lainnya, seperti meletakkan «Alamat» di baris bawahnya.

Langkah 4: Preview & Finish

Pratinjau Hasil dan Eksekusi Akhir

Sebelum mencetak ratusan kertas, sangat penting untuk memeriksa apakah datanya masuk dengan benar.

  • Preview Results (Pratinjau): Klik tombol Preview Results di tab Mailings. Kode «Nama_Peserta» akan seketika berubah menjadi nama asli (misal: Budi Santoso). Gunakan tombol panah (kanan/kiri) di sebelahnya untuk melihat data orang kedua, ketiga, dan seterusnya.
  • Finish & Merge: Jika semua sudah benar, klik tombol ini untuk eksekusi.
    • Pilih Print Documents... jika komputer sudah terhubung ke printer untuk langsung mencetak semuanya.
    • Pilih Edit Individual Documents... jika Anda ingin Word membuat file baru berisi ratusan halaman (setiap halaman berisi nama berbeda) agar Anda bisa mengecek ulang atau menyimpannya sebagai file gabungan (PDF).
