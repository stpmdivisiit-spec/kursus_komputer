Modul Spesialisasi Pencarian Data

Master Lookup & Referensi Excel

Kuasai teknik menarik data antar tabel secara instan. Modul ini fokus secara eksklusif pada pemahaman mendalam tentang VLOOKUP, HLOOKUP, dan fungsi modern XLOOKUP.

Mulai Simulasi Interaktif

3 Pilar Fungsi Referensi Data

VLOOKUP

Pencarian Vertikal

Digunakan ketika Tabel Master Anda memanjang ke bawah (header berada di baris atas). VLOOKUP akan mencari nilai kunci di kolom paling kiri, lalu mengambil hasil di kolom sebelah kanannya.

HLOOKUP

Pencarian Horizontal

Digunakan ketika Tabel Master Anda memanjang ke samping (header berada di kolom paling kiri). HLOOKUP akan mencari kunci di baris paling atas, lalu mengambil hasil di baris bawahnya.

XLOOKUP

Pencarian Modern Fleksibel

Generasi terbaru pengganti VLOOKUP/HLOOKUP. Bebas mencari ke arah kiri maupun atas, tidak perlu menghitung nomor kolom, dan sudah memiliki fitur anti-error (pengganti IFERROR) bawaan.

Anatomi Rumus & Aturan Emas

Aturan Emas: KUNCI TABEL MASTER (F4)

Kesalahan paling umum yang membuat rumus Lookup error menjadi #N/A saat diseret (di-copy) ke bawah adalah karena rentang tabel master bergeser.
Solusi Mutlak: Setelah menyorot/blok Tabel Master, Anda wajib menekan tombol F4 di keyboard satu kali hingga muncul simbol Dolar ($).
Contoh: F3:H5 menjadi $F$3:$H$5.

Syntax VLOOKUP
=VLOOKUP(
  key,
  tabel,
  col_num,
  0
)
  • key Nilai yang mau dicari (Sel acuan).
  • tabel Blok Master Data (Wajib F4).
  • col_num Urutan kolom hasil (Angka 1, 2, 3..).
  • 0 / FALSE Agar pencarian spesifik/akurat.
Syntax HLOOKUP
=HLOOKUP(
  key,
  tabel,
  row_num,
  0
)
  • Parameter 1, 2, dan 4 sama persis dengan VLOOKUP.
  • row_num Hitung urutan Baris dari atas ke bawah untuk mengambil data (Angka 1, 2, 3..).
Syntax XLOOKUP
=XLOOKUP(
  key,
  array_cari,
  array_hasil,
  "jika_error"
)
  • key Nilai yang mau dicari.
  • array_cari Blok 1 kolom saja tempat key berada (F4).
  • array_hasil Blok 1 kolom hasil (F4).
  • jika_error (Opsional) Teks jika tidak ketemu.
