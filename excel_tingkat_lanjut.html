1. Pivot Table Lanjutan

Calculated Field, Grouping, & Pivot Chart

Pemula menggunakan Pivot Table hanya untuk sekadar drag-and-drop dan men-sum data. Seorang ahli menggunakannya untuk membedah data lebih dalam tanpa harus menyentuh atau menambah kolom pada data mentah aslinya.

Membuat rumus di dalam Pivot Table
  • Calculated Field (Bidang Kalkulasi)

    Bagaimana jika data mentah Anda hanya punya kolom "Pendapatan" dan "HPP (Modal)", tapi Anda ingin Pivot menampilkan "Keuntungan"? Jangan tambahkan kolom baru di data mentah!
    Gunakan PivotTable Analyze > Fields, Items, & Sets > Calculated Field. Anda bisa membuat rumus = Pendapatan - HPP langsung di dalam Pivot. Ini sangat efisien dan memperingan ukuran file.

  • Grouping Data (Pengelompokan Otomatis)

    Jika Anda menarik kolom "Tanggal" ke Rows, Pivot Table versi modern otomatis akan mengelompokkannya menjadi Bulan dan Tahun. Namun Anda juga bisa mengelompokkan Angka.
    Contoh: Mengelompokkan kolom "Umur Pelanggan" ke dalam rentang (20-29, 30-39, 40-49) cukup dengan klik kanan angka di Pivot > Group... > Atur Starting at dan By (jarak).

  • Pivot Chart

    Grafik yang secara langsung terhubung ke Pivot Table Anda. Jika Anda mem-filter Pivot Table, grafiknya akan ikut bergerak dan beranimasi secara real-time.

2. Power Query WAJIB KUASAI

ETL (Extract, Transform, Load) - Pembersih Data Otomatis

Jika Anda hanya boleh belajar satu fitur tingkat lanjut di Excel, pelajarilah Power Query (Tab Data > Get Data). Ini adalah revolusi terbesar Excel dalam 10 tahun terakhir untuk urusan *Data Cleaning* dan rekonsiliasi laporan mingguan/bulanan.

Mengapa Power Query Sangat Luar Biasa?
  • Import dari Segala Sumber: Tarik data dari ratusan file CSV sekaligus, dari Website, PDF, Database SQL, atau sekadar Folder di komputer Anda.
  • Unpivot Data: Mengubah format tabel laporan bentuk matriks (yang melebar ke kanan per bulan) menjadi tabel tabular ke bawah (database siap olah) hanya dengan 2 klik.
  • Merge Queries: Menggantikan VLOOKUP secara permanen! Menggabungkan 2 tabel besar menjadi 1 secara visual jauh lebih cepat dan tidak membebani memori komputer.
Merekam Langkah (Applied Steps)

Keajaiban sesungguhnya adalah: Power Query merekam setiap klik yang Anda lakukan (hapus kolom kosong, pecah nama, ubah format tanggal).

Bulan depan, saat Bos mengirim file data mentah yang baru, Anda TIDAK PERLU melakukan pembersihan manual lagi. Cukup klik Refresh All, dan Power Query akan mengulangi ratusan langkah pembersihan itu secara otomatis dalam hitungan detik!

3. Power Pivot (Data Model)

Relational Database & DAX (Menembus Limit 1 Juta Baris)

Satu sheet Excel hanya bisa memuat maksimal 1.048.576 baris. Jika perusahaan Anda punya 5 juta baris data transaksi, Excel biasa akan lumpuh/crash. Power Pivot menyimpannya di mesin kompresi di balik layar Excel.

  • Relasi Antar Tabel (Relationships)

    Daripada menggabungkan Tabel Transaksi dan Tabel Master Barang dengan ratusan ribu rumus VLOOKUP (yang membuat file Excel Anda berukuran 500MB dan super lemot), Anda cukup menarik garis relasi (Relationships) antara Kode Barang di kedua tabel tersebut secara visual. Pivot Table Anda nantinya bisa mengambil data dari dua tabel berbeda sekaligus!

  • DAX Dasar (Data Analysis Expressions)

    Bahasa rumus tingkat dewa yang juga digunakan di Power BI. Berbeda dengan rumus Excel biasa yang bekerja pada sel (A1), DAX bekerja pada kolom secara keseluruhan.

    Total Sales := SUMX(Sales, Sales[Qty] * Sales[Price])
    Sales YTD := TOTALYTD([Total Sales], 'Calendar'[Date])

    *Rumus DAX di atas bisa menghitung total penjualan berjalan dari awal tahun (YTD) tanpa harus membuat tabel pivot manual per bulan.

4. Dashboard & Visualisasi

Membuat Laporan Interaktif Eksekutif

Sebuah *Dashboard* adalah gabungan dari beberapa Pivot Chart yang disatukan dalam satu layar yang rapi, bersih tanpa *gridlines*, dan sangat interaktif layaknya aplikasi.

  • Combo Chart (Grafik Kombinasi)

    Menggabungkan dua jenis grafik dalam satu kanvas, dan memiliki dua sumbu Y (Kiri dan Kanan).
    Contoh: Grafik Batang (Bar) untuk menunjukkan jumlah unit terjual (skala ribuan), digabungkan dengan Grafik Garis (Line) untuk persentase Profit Margin (skala 0%-100%).

  • Slicer & Timeline (Tombol Interaktif)

    Slicer adalah filter visual berbentuk tombol keren. Jika Anda memiliki 4 grafik di dashboard Anda, Anda bisa menghubungkan 1 Slicer "Cabang" ke 4 grafik tersebut (menggunakan Report Connections).
    Saat atasan Anda mengeklik tombol "Cabang Bali", keempat grafik akan beranimasi dan merubah datanya secara serentak khusus untuk Bali.

5. Array Formula / Dynamic Array

Revolusi Rumus Excel (Tersedia di Excel 365 / 2021+)
Spill Behavior: Pada Excel tradisional, 1 rumus = 1 hasil di sel tersebut. Dengan Dynamic Arrays, Anda mengetik 1 rumus di sel A1, hasilnya bisa "tumpah" (Spill) menyebar otomatis memenuhi sel ke bawah dan ke samping menyesuaikan banyaknya data!

  • =FILTER()

    Menyaring data tanpa perlu menekan menu filter. Seperti contoh tabel di atas, rumus diketik di 1 sel, dan secara magis hasilnya tumpah meniru tabel aslinya khusus untuk kriteria "Jakarta".

  • =SORT()

    Mengurutkan data tabel array secara dinamis tanpa menu Sort manual. Data akan selalu terurut sesuai abjad/angka meski data mentah aslinya berubah.

  • =UNIQUE()

    Menggantikan Remove Duplicates. Jika diketik =UNIQUE(C2:C6), rumus ini akan langsung menghasilkan "Jakarta, Bali, Bandung" secara tumpah ke bawah tanpa menghapus data aslinya. Sangat sempurna digabungkan sebagai referensi untuk Data Validation Dropdown menggunakan tanda pagat (#), misal: =$E$2#.

6. Automation Advanced

VBA (Makro) dan Office Scripts

Ketika rumus dan fitur bawaan Excel sudah tidak mampu memenuhi *workflow* unik perusahaan Anda, saatnya memprogram Excel Anda sendiri.

VBA Lanjutan (Visual Basic for Applications)

Bahasa pemrograman legendaris Excel. Anda bisa membuat tombol yang jika diklik akan memproses data, membuat ratusan file PDF baru, lalu mengirimkannya via Outlook ke masing-masing email klien secara otomatis.

Sub KirimEmailOtomatis()
  Dim lastRow As Integer
  lastRow = Cells(Rows.Count, 1).End(xlUp).Row
  For i = 2 To lastRow
    If Cells(i, 3).Value = "Lunas" Then
      'Kirim PDF ke email
    End If
  Next i
End Sub
Office Scripts (Excel Modern / Web)

Kelemahan VBA adalah: file .xlsm tidak bisa dijalankan di Excel Online (Web) maupun HP. Sebagai solusi masa depan, Microsoft memperkenalkan Office Scripts berbasis bahasa TypeScript/JavaScript.

function main(workbook: ExcelScript.Workbook) {
  let sheet = workbook.getActiveWorksheet();
  let range = sheet.getRange("A1:C10");
  range.getFormat().getFill().setColor("yellow");
}

*Office Scripts dapat dihubungkan dengan Power Automate, sehingga Anda bisa membuat alur: "Jika ada email masuk dengan lampiran Excel, otomatis ekstrak datanya ke file Master Excel di SharePoint".
