Otomatisasi Level Selanjutnya

Di tingkat menengah, Anda tidak sekadar menjumlahkan angka. Anda akan belajar memotong teks (ID Karyawan), menghitung umur otomatis, membuat aturan validasi data, hingga melakukan proyeksi skenario bisnis dengan What-If Analysis.

1. Fungsi Teks (Memanipulasi Karakter)

LEFT, RIGHT, MID, LEN, TRIM, CONCAT

Fungsi teks berguna untuk mengekstrak atau menggabungkan data string (teks). Sangat berguna saat berhadapan dengan Kode Transaksi, NIK, atau Nomor Resi.

  • LEFT, RIGHT, MID: Mengambil beberapa karakter dari Kiri, Kanan, atau Tengah teks.
  • LEN: Menghitung total jumlah karakter (termasuk spasi). Berguna untuk mengecek validitas NIK (harus 16 digit).
  • TRIM: Menghapus spasi ganda yang tidak sengaja terketik di awal, tengah, atau akhir kalimat. Sangat ampuh untuk *data cleaning*.
  • CONCAT / CONCATENATE: Menggabungkan dua atau lebih teks dari sel yang berbeda menjadi satu. (Alternatif: pakai simbol &).
fx
=MID(A3, 5, 4)

1 Studi Kasus: Ekstrak Kode Karyawan
2 MKT-2021-A1 MKT
=LEFT(A2, 3)		 2021
=MID(A2, 5, 4)		 A1
=RIGHT(A2, 2)
3 FIN-2023-B2 FIN 2023 B2
4 Contoh TRIM dan CONCAT
5 " Budi Santoso " =TRIM(A5) hasilnya "Budi Santoso" (spasi ekstra hilang).
6 Budi =CONCAT(A6, " ", D3) hasilnya "Budi B2".

Penjelasan MID: =MID(A3, 5, 4) artinya ambil karakter dari sel A3, mulai dari karakter ke-5, dan ambil sebanyak 4 karakter ke kanan.

2. Fungsi Tanggal

TODAY(), NOW(), DATEDIF

Sangat krusial untuk departemen HR (menghitung masa kerja) atau Finance (menghitung jatuh tempo tagihan).

  • =TODAY()
    Menampilkan tanggal hari ini. Akan otomatis berubah setiap hari saat file dibuka. Tidak butuh argumen di dalam kurung.
  • =NOW()
    Sama seperti TODAY, tetapi menyertakan jam dan menit saat ini.
  • =DATEDIF()
    Rumus tersembunyi Excel untuk menghitung selisih (difference) antara dua tanggal. Sangat akurat untuk menghitung umur atau masa kerja (Tahun/Bulan/Hari).
fx
=DATEDIF(B3, $C$1, "Y")

3 Sisca 10/05/1995 29

Struktur DATEDIF: =DATEDIF(tgl_awal, tgl_akhir, "kode_waktu")
Ganti kode waktu dengan: "Y" (Tahun/Year), "M" (Bulan Total), "D" (Hari Total).
Contoh di atas: DATEDIF(B3, $C$1, "Y") menghitung tahun penuh antara tanggal lahir dan hari ini.

3. Logical Advanced

IF Bertingkat dan Kombinasi AND/OR

Bagaimana jika kondisinya tidak hanya "Lulus/Gagal", tetapi ada Grade A, B, C, D? Anda butuh IF Bertingkat (Nested IF) atau menggunakan rumus modern IFS.

A (Siswa) B (Nilai) C (Absen) D (Grade - IF Bertingkat) E (Beasiswa - IF+AND)
1 Joko 85 100% B Layak
2 Rina 92 80% A Ditolak
Rumus Kolom D (Nested IF)

Memberi Grade: A (>90), B (>80), C (>70), Sisanya D.

=IF(B1>90, "A", IF(B1>80, "B", IF(B1>70, "C", "D")))

*Excel 2019+ bisa pakai IFS:
=IFS(B1>90,"A", B1>80,"B", B1>70,"C", TRUE,"D")

Rumus Kolom E (IF + AND)

Syarat Beasiswa: Nilai harus >=85 DAN Absen harus 100%.

=IF(AND(B1>=85, C1=1), "Layak", "Ditolak")

*Baris 2 (Rina) Ditolak karena Absennya tidak 100%, meski nilainya A.

4. Data Validation

Mencegah Salah Ketik dengan Dropdown & Restriksi

Pernahkah Anda merekap data di mana orang mengetik "Lunas", "lunas", "LUNAS", atau Typo "Luans"? Ini merusak rekapitulasi. Gunakan alat di menu Data > Data Validation untuk memaksakan aturan input.

  • Dropdown List (Daftar Pilihan)

    Membuat sel hanya bisa diisi dari opsi yang disediakan (klik panah bawah).

    Cara: Pilih Allow: List. Di bagian Source, ketik: Lunas,Belum Lunas,Cicil.

    Status: Lunas
  • Input Restriction (Pembatasan Input)

    Mencegah input nilai yang tidak masuk akal.
    - Whole Number: Batasi umur karyawan hanya boleh antara angka 18 hingga 60.
    - Text Length: Batasi input Nomor Telepon maksimal 13 karakter. Jika user mengetik lebih panjang, Excel akan memunculkan pop-up Error dan menolak ketikan tersebut.

5. Conditional Formatting (Lanjutan)

Mewarnai Baris Secara Otomatis Berdasarkan Rumus

Di tingkat dasar, Anda mewarnai sel merah jika angkanya di bawah 70. Di tingkat menengah, Anda bisa mewarnai SATU BARIS PENUH jika kolom statusnya berubah menjadi "Selesai" menggunakan Formula-based formatting.

A (Tugas) B (PIC) C (Status)
1 Kirim Laporan Pajak Ani Selesai
2 Revisi Anggaran Budi Proses
3 Gaji Karyawan Dani Terlambat
Cara Mewarnai Seluruh Baris (Contoh warna Hijau "Selesai"):
1. Blok seluruh tabel data (A1:C3).
2. Ke Home > Conditional Formatting > New Rule > Use a formula to determine which cells to format.
3. Ketik rumus: =$C1="Selesai". (Perhatikan tanda Dolar $ hanya di depan huruf C. Ini mengunci pengecekan ke kolom C, tapi membiarkan baris bergerak).
4. Klik Format, pilih warna Fill Hijau.

6. What-If Analysis

Data Table & Scenario Manager

Alat tingkat lanjut di tab Data untuk melakukan proyeksi atau simulasi bisnis tanpa merusak rumus asli Anda. Sangat disukai oleh para Analis Keuangan (Financial Analyst).

Data Table (Tabel Matriks)

Digunakan untuk melihat semua kemungkinan hasil jika Anda mengubah 1 atau 2 variabel secara bersamaan.
Contoh Kasus: Simulasi Cicilan KPR (Rumus PMT). Daripada mengetik ulang rumus bunga bank 5%, 6%, 7% satu per satu, Data Table otomatis mengisi kotak matriks perbandingan suku bunga dan lama tahun cicilan.

Cicilan /Bulan 10 Tahun 15 Tahun
Bunga 5% 5.300.000 3.950.000
Bunga 7% 5.800.000 4.400.000

*Angka di dalam matriks dihasilkan otomatis oleh Data Table.

Scenario Manager

Digunakan jika Anda punya lebih dari 2 variabel yang berubah, misalnya proyeksi Laba/Rugi tahun depan. Anda dapat membuat dan menyimpan beberapa set "Skenario" di file yang sama.

  • Skenario Pesimis (Worst Case): Penjualan turun 20%, Biaya Iklan naik 10%.
  • Skenario Normal (Base Case): Angka penjualan standar saat ini.
  • Skenario Optimis (Best Case): Penjualan naik 50%, Biaya Iklan turun 5%.

Anda cukup mengklik nama Skenario di menu Scenario Manager, dan seluruh angka beserta grafik/chart di layar Anda akan seketika berubah mengikuti proyeksi tersebut. Tersedia juga fitur "Summary" untuk membandingkan ketiganya dalam satu tabel ringkasan (mirip Pivot Table).
